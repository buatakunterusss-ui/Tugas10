Tentang OrangDesaJugaNgertiDolar

Sistem Klasifikasi Nominal Uang Kertas Rupiah Berbasis CNN

Deskripsi Sistem

OrangDesaJugaNgertiDolar adalah aplikasi web berbasis kecerdasan buatan yang mampu mengklasifikasikan nominal uang kertas Rupiah Indonesia secara otomatis dari gambar yang diunggah pengguna.

Sistem ini menggunakan arsitektur MobileNetV2 dengan pendekatan Transfer Learning dari ImageNet, yang kemudian di-fine-tune menggunakan dataset gambar uang kertas Rupiah emisi terbaru. Dengan arsitektur yang ringan namun akurat, model ini cocok untuk dijalankan di lingkungan web maupun perangkat mobile.

Arsitektur Model CNN
1
Input Layer
Gambar RGB 224×224 piksel, dinormalisasi ke [0,1]
2
MobileNetV2 Base (Frozen → Fine-tuned)
154 layer konvolusi depthwise separable, pretrained ImageNet
3
GlobalAveragePooling2D
Meratakan feature map menjadi vektor 1D
4
Dense (256) + BatchNorm + Dropout 0.4
Fully connected layer dengan regularisasi
5
Dense (128) + Dropout 0.3
Lapisan fully connected kedua
6
Output Layer – Softmax (7 kelas)
Rp1.000 / Rp2.000 / Rp5.000 / Rp10.000 / Rp20.000 / Rp50.000 / Rp100.000
Teknologi yang Digunakan
  • TensorFlow / Keras — Training model CNN
  • MobileNetV2 — Arsitektur Transfer Learning
  • Flask — Backend web framework Python
  • Bootstrap 5 — Frontend UI/UX
  • scikit-learn — Evaluasi model (confusion matrix)
  • Kaggle Dataset — Rupiah Banknotes Dataset
